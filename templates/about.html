Fake news spreads misinformation and can cause serious damage. This project uses Natural Language Processing (NLP) and a machine learning classifier hosted in a Flask backend to detect misleading or fake articles.

ML Workflow

1. Data Collection: Gather labeled real & fake news datasets.
2. Text Preprocessing: Tokenization, stopword removal, stemming/lemmatization, TF-IDF or embeddings.
3. Model Training: Train classifiers (Logistic Regression / SVM / Transformer-based models).
4. Prediction & Confidence Scoring: Model returns label + confidence score.

Features

  • Paste text or URL to detect fake news
  • Confidence score instead of binary-only output
  • Highlight suspicious words
